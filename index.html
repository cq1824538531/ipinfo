❓ 常见问题

IP 归属地查询结果准确吗？

IP 归属地基于运营商 IP 段分配，通常精确到城市级别。使用 VPN、代理或移动网络时可能存在偏差。

内网 IP 能查到归属地吗？

不能。192.168.x.x、10.x.x.x 等私有地址仅在局域网内使用，不具备公网地理位置信息。

查询频率有限制吗？

免费 API 每分钟最多支持 45 次查询，个人日常使用完全足够。如需批量查询请参考帮助中心。
