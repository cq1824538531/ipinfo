🔒 隐私政策

我们重视并保护您的隐私与数据安全

📅 最后更新：2025年5月

📋 概述

感谢您使用 IP 归属地查询工具。本隐私政策说明我们如何处理您在使用本服务过程中产生的信息。我们承诺保护您的隐私安全，并以最小化数据收集为原则运营本服务。

📊 信息收集

我们处理的信息

  • 查询的 IP 地址：您在输入框中填写的 IP 地址，仅用于向第三方 API 发起查询，不会被本服务器存储。
  • 您的访问者 IP：当您点击「查询我的 IP」时，我们会将您的公网 IP 发送给第三方 API 获取地理位置数据，该过程不经过我们的服务器。
  • 匿名访问统计：我们可能通过 Google Analytics 等工具收集匿名访问量数据（页面浏览量、来源渠道等），不含任何个人身份信息。

🎯 信息使用

我们收集的有限信息仅用于：提供 IP 归属地查询服务、改善用户体验、统计匿名访问量以优化服务质量，以及维护网站安全。我们不会将您的信息出售、出租或提供给任何第三方用于营销目的。

🔐 数据保护措施

  • 全站部署 HTTPS，所有请求均经过 SSL/TLS 加密传输。
  • 本工具为纯前端应用，IP 查询请求直接发送至第三方 API，不经过我们的服务器，我们无法看到您的查询记录。
  • 不设置任何追踪型 Cookie，不建立用户画像。

🌐 第三方服务

本工具使用 IPinfo.io 提供 IP 地理位置数据。当您发起查询时，IP 地址将发送至 IPinfo.io 服务器。建议您查阅其 隐私政策 以了解详情。

🍪 本地存储

本工具不使用任何 Cookie 或本地存储（localStorage）保存您的查询记录或偏好设置。您可以通过浏览器设置管理本站的存储权限。

👤 您的权利

  • 随时停止使用本服务。
  • 要求了解我们如何处理您的信息。
  • 如您认为本工具侵犯了您的隐私权益，请通过 联系我们 页面反馈。

🔄 政策更新

本隐私政策可能随服务更新而修订。重大变更时，我们会在网站显著位置发布通知。建议您定期查看本页面。

重要提示：使用本网站即表示您已阅读并同意本隐私政策。如不同意，请停止使用本服务。
